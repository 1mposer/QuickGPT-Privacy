Privacy Policy for QuickGPT Chrome Extension

QuickGPT does not collect, store, or share personal information from users.

The extension uses OpenAI's API to process highlighted text directly and securely. No text data is logged or retained by the extension.

By using QuickGPT, you consent to the transmission of highlighted text to the OpenAI API for the sole purpose of generating responses.

If you have any questions or concerns, please contact us at your-email@example.com.
